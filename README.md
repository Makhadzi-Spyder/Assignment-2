# 🌸 Myth-sterious App 🌸

## 🌿 Overview 🌿
The Myth-sterious App is an Android application developed using Kotlin in Android Studio.  
The purpose of this app is to help users to test their knowledge of common life hacks and myths by showing them flashcards of said hacks and they determine if it is true or not by clicking 'true' or 'false'

The app uses a flashcard-style approach where each life hack is presented individually. The user selects an answer, true or false, receives immediate feedback, and their progress is tracked. At the end, they are able to see their score as well as an appropriate message for how much they got and they get to review all the hacks they were tested on to see why they are right or wrong.

---

## 🌷 Features  🌷
- Flashcard-style life hack questions
- True and False answer buttons
- Immediate feedback after each answer
- Score tracking system
- Results screen displaying final score
- Personalised feedback based on performance
- Review section with explanations for each life hack

## 🌼 How the App Works 🌼
1. The user is greeted on the welcome page
2. The user then pressed the **Start** button and is taken the FlashCards screen.
3. The user is presented with a life hack statement on the FlashCards screen.
4. The user clicks on either **True** or **False**.
5. The app checks then displays feedback (“Correct” or “Wrong”).
6. The user presses the **Next** button to move to the next question.
5. The app loops through all the statements until they have all been answered.
8. The app then navigates to the Score screen.
9. The user’s score and feedback are displayed.
10. The user can press the **Review** button to view explanations for all life hacks even the ones they got right.

---

## 🌸 Usage Example 🌸
Example interaction:
- Life hack: *“Cold showers boost alertness”*
- User selects: **True**
- Feedback: *Correct!*
- User continues through quiz
- Final result: *“Your score: 7 / 11 – Well done!”*
